Introduction

This is a project to create an electronic/computer chess board. I hope many people will copy this projects. To achieve this, I wanted to design a chess board that was as simple as possible (low cost), easy to build, and easy to program in the firmware. Also this chess board is open software and open hardware.

This is one of the simplest electronic chess boards, which is why I named this project a didactic chess board. This project can be used as a didactic tool before someone can design a more sophisticated chess board.

Chess is a board game that has been around for thousands of years. Since the advent of computer games, chess' popularity has waned. Will people no longer play chess? Apparently not, as many people still play chess today. Digital technology seems to be preventing the game from disappearing.

Today, we can play chess online with people around the world 24 hours a day. lichess.org is a free chess server, without even advertising. Around 40,000 people play chess on Lichess at any given time. Over a million chess games are played on Lichess every day. It's incredibly expensive to run, but free to users. This is made possible through donations and the open-source Lichess program, which is powered by volunteers.

Chess is essentially a board game with wooden pieces, but online chess games have shifted the game from the board to a computer or smartphone screen. The pieces are moved using a mouse or touchscreen. It's no surprise that today we find young people who play chess but never use a board. They always play online or against computer. Even now, local and international chess championships still use real boards.

 Since February 2020, Lichess officially launched the Board API (Application Programming Interface), which allows people to play chess using a real board when playing chess online on Lichess. However, after six years, to my knowledge, there is no chess board that is easy to build, open source, and open hardware like the Didactic Chess Board. Therefore, I hope the Didactic Chess Board will become the "official Lichess board." If we search the GitHub site, there are chess boards that are open source, but not open hardware. We have to buy ready-made boards from the developers. There are also open source and open hardware boards, but difficult to build.

 If you want to play chess using a real board when playing online or playing chess against the computer and want to make it yourself (DIY), then this project is suitable for you.
